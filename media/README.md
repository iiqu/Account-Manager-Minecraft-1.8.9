# Media Files

This directory contains screenshots, videos, and GIFs showcasing the Account Manager mod functionality.

## Directory Structure

- `screenshots/` - Static images showing the mod interface and features
- `videos/` - Video demonstrations (MP4 format)

## Adding Media

When adding new media files:
- Use descriptive filenames (e.g., `account-manager-gui.png`, `anonymous-mode-demo.gif`)
- Keep file sizes reasonable for GitHub viewing
- Update the main README.md to reference new media files
